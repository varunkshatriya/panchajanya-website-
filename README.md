# Panchajanya Website

A modern, responsive, and high-performance frontend website for **Panchajanya**—a pioneering manufacturer and marketer of animal feed supplements, cleaning products, livestock solutions, and human wellness products with over 25 years of trust and quality.

Built using **Next.js**, **Tailwind CSS**, and **Framer Motion**, this application features premium interactive designs, micro-animations, count-up stats, and a modern layout.

---

## 🚀 Technologies Used

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router, TypeScript)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Charts/Data Vis**: [Recharts](https://recharts.org/)

---

## 🛠️ Getting Started

### Prerequisites

Make sure you have Node.js installed on your system.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/varunkshatriya/panchajanya-website-.git
   cd panchajanya-website-
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) (or the port specified in the console) in your browser to see the result.
